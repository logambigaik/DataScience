Tools & Techniques: Seaborn -Visualising the Data ▶
Histogram 
import matplotlib.pyplot as plt 
import seaborn as sns 
df.hist(figsize=(10,8))
plt.show()
Description:

  • This line of code generates histograms for all numerical columns in the DataFrame df.
  • hist() is a Pandas method that creates a histogram for each numeric feature (like integers or floats).
  • figsize=(10,8) sets the size of the entire plot figure — 10 units wide by 8 units tall.

Output:
Histogram 
df[["IMF_Estimate", "UN_Estimate", "WorldBank_Estimate"]].hist(bins=3, figsize=(12,9))
plt.show()
Description:

This command creates histograms for three specific columns in the DataFrame

  • df[["IMF_Estimate", "UN_Estimate", "WorldBank_Estimate"]]: Selects only the three columns from the DataFrame.
  • .hist(...): Plots a histogram for each selected column.
  • bins=3: Divides the data range into 3 bins or intervals. This controls how coarse or fine the histogram
  • figsize=(12,9): Sets the size of the entire plot figure (12 units wide, 9 units tall).

Output:

Description:

sns.heatmap() function from the Seaborn library is used to create a heatmap, which is a two-dimensional graphical representation of data where individual values are represented by color intensity.

Output:
HeatMap 
corr = df[["IMF_Estimate", "UN_Estimate", "WorldBank_Estimate"]].corr()
plt.figure(figsize=(9,6))
sns.heatmap(corr, annot=True, fmt=".2f", cmap = 'GnBu', annot_kws={"size": 12})
# annot show labes, cmap change colour, annot_kws change text size, fmt decimal
plt.show()
Description:
  • corr: This is the correlation matrix generated by df.corr(). It contains pairwise correlation values (ranging from -1 to 1) between all numerical columns.
  • annot=True: This tells Seaborn to annotate each cell of the heatmap with its numeric correlation value, making it easier to interpret the strength of relationships.
