Replace 0 values with nulls (NaN) in IMF_Estimate

df["IMF_Estimate"] = df["IMF_Estimate"].replace(0, np.nan)

This helps to avoid skewing averages and better reflects missing data.

Calculate average of WorldBank and UN estimates

df['avg_worldbank_un'] = df[['WorldBank_Estimate', 'UN_Estimate']].mean(axis=1)

Create a new column that averages the two estimates for a more robust estimate.

Fill missing IMF_Estimate values with the calculated average

df["IMF_Estimate"] = df["IMF_Estimate"].fillna(df['avg_worldbank_un'])

This fills IMF missing values using the combined average from WorldBank and UN data.
